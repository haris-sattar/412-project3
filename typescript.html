What is TypeScript and what are its advantages in Web Development?

In basic terms, TypeScript is a better JavaScript. TypeScript has all the same features that JavaScript has, but it adds static typing, which allows developers to add types. This means if you were to code in JavaScript, it would still understand it. So, it is easier to catch errors in TypeScript since you have to specify which type is being passed around, something that is not done in JavaScript. It does this because TypeScript does compile time type checking. This means it checks the types before running the code, not when you run it. Even though it has these great features, it can still run in node.js or in a browser.
